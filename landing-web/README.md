# Crema Landing Page

The public landing page for Crema, built with Next.js and Tailwind CSS v4.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Tech Stack

- Next.js 16 (App Router)
- Tailwind CSS v4
- Fonts: Climate Crisis, Inter (via `next/font/google`)
